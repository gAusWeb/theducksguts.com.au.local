TK Shortcodes is a simple plugin which enables you to add beautiful and useful shortcodes to your content.

== Available shortcodes ==

* Button
* Row
* Dropcap
* Icon
* Columns
* Fullwidth section
* Infobox
* Toggle
* Tabs
* Accordion
* Progress bar
